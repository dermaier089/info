Beziehungen:
SPIEL enthält 1 FAENGER (Referenzattribut)
SPIEL enthält n BALL (Feld/Array)

Kapselung: Attribute sind private — Zugriff nur über Methoden (Getter/Setter).
baelle: Ball[] — Das Feld im Überblick
So funktioniert das Feld:
• baelle = new Ball[5] erzeugt 5 Plätze
• Jeder Platz speichert eine Referenz auf ein Ball-Objekt
• baelle[i] = null → Platz ist leer (Ball gefangen)
• baelle.length = 5 (Größe des Feldes)

foo() prüft für jeden Platz:
1. Ist der Platz nicht null?
2. Berührt der Ball den Fänger?
3. Wenn ja → Ball entfernen, Platz auf null setzen
